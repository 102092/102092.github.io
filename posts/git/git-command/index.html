Update commit author#

  1. 수정하고 싶은 직전 커밋의 Hash 확인

  2. git rebase

git rebase -i -p 커밋hash
  • 위 명령어 입력할 경우, 해당 해쉬 커밋 이후 부터 모든 커밋이 리베이스 대상이됨.

  1. rebase 모드에서, 수정하고자하는 커밋 상태를 e 로 변경

  2. author 수정

git commit --amend --author="사용자명 <이메일>"
  1. rebase –continue
git rebase --continue
